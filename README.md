# Billion InnoTech Landing Page

This repository contains a premium dark-themed landing page for Billion InnoTech, built with Next.js, Tailwind CSS, and GSAP.

## Features

- Modern futuristic landing page design
- Dark Apple/Tesla-inspired visual style
- Smooth scroll-based reveal animations
- Responsive layout for desktop and mobile
- Hero section, showcase section, services section, and contact section

## Tech Stack

- Next.js
- React
- Tailwind CSS
- GSAP
- TypeScript

## Run Locally

```bash
cd /Users/admin/billion-innotech-site
npm install
npm run dev
```

Then open:

```text
http://127.0.0.1:3003
```
